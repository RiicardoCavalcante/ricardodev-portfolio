Habilidades

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo.

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a apresentação e o layout de documentos HTML na web. Ele permite definir cores, fontes, espaçamentos e outros estilos, separando a estrutura do conteúdo, facilitando a criação de designs atraentes e consistentes em páginas da web. As regras CSS consistem em seletores e declarações, proporcionando controle modular sobre elementos específicos. A abordagem "em cascata" permite a sobreposição de estilos, garantindo flexibilidade e adaptação a diferentes dispositivos e necessidades de design.

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.
